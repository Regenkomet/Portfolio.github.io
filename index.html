Meine Programmier Erfahrung

Meine Ersten Schritte am Computer habe ich früh gemacht, zuerst allerdings ohne mich für die Technik dahinter zu interessieren. Das Intresse an der Informatik kam zuerst bei einem Wahlppflicht Kurs in der Schule, dort habe gemerkt wie vielfältig das Programmieren sein kann. Begonnen habe ich dann mit HTML und CSS, welche ja genauer genommen keine Programmiersprachen sind. Weiter ging es dann mit Java bzw. Javascript, alle auch hier erstellten Projekte wurden mithilfe von HTML, CSS und Javascript erstellt. Zusätzlich habe ich auch schon in die Welt von Python hineingeschnuppert. Auch in die Welt von React habe ich mich gewagt und versuche das mithilfe von Anleitungen im Internet weiter zu erlernen. In der Zukunft möchte ich dann sowohl Frontend als auch Backend gut beherrschen.
